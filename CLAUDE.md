# Claude Code 지침

## 언어 설정
- **모든 응답을 한국어로 작성하세요.**
- 코드 주석이나 문서화도 가능한 경우 한국어로 작성하세요.
- 기술 용어는 필요에 따라 영어를 병행 사용할 수 있습니다.

## 응답 스타일
- 명확하고 간결하게 설명하세요.
- 전문적이고 객관적인 톤을 유지하세요.
- 사용자의 질문에 정확하게 답변하세요.

## 코드 작성 원칙
- 이미 존재하는 파일을 수정하는 것을 우선하고, 꼭 필요한 경우에만 새 파일을 생성하세요.
- 보안 취약점(명령어 인젝션, XSS, SQL 인젝션 등)을 방지하세요.
- 과도한 엔지니어링을 피하고 요청된 작업에만 집중하세요.
